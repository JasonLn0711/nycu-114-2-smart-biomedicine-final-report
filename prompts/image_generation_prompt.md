# Image Generation Prompt

Use `$ian-xiaohei-illustrations`.

請根據 `docs/storyboard.md`，為智慧生醫概論期末報告影片產生 `16:9` 插圖素材。

要求：

1. 每個 Scene 產生一張 `16:9` PNG。
2. 輸出到 `public/images/`。
3. 檔名使用 `scene_001.png`, `scene_002.png`。
4. 每張圖只表達一個核心概念。
5. 角色使用小黑 / 小拳風格，但避免畫面文字過多。
6. 醫療、AI、病患流程、風險分流、資料流向要清楚。
7. 同時更新 `docs/asset_manifest.md`，記錄每張圖對應 scene、用途、prompt 摘要與生成狀態。

風格：

- 純白背景。
- 黑色手繪線稿。
- 大量留白。
- 少量台灣繁體中文手寫批註。
- 不要 PPT、商業插畫、幼稚可愛、正式流程圖、複雜架構圖或真實 UI 截圖。
- 不要在左上角寫圖名或結構類型標題。
